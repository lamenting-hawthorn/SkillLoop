1. Agent Architecture: Live Runtime Invocation

Incoming Message

User sends text through CLI, WhatsApp, Telegram, or API.

Adapter normalizes the channel payload.

Resolve Actor

Map sender to actor_id, org_id, role, and session_id.

Production gate: no demo identity fallback.

Store Event

Append raw event to event_store.events.

Observation before memory.

Invoke LangGraph

Run the workflow with thread_id checkpointing.

graph.invoke(state, config)

Answer + Writeback

Retrieve memory, call model, decide salience, write durable memory.

Current graph is linear MVP.

2. Agent Architecture: LangGraph Workflow

resolve_identity

Attach actor and trace_id to the run.

retrieve_memory

Search Hermes native memory first, then Postgres hybrid retrieval, then graph expansion.

call_model

Construct prompt with retrieved context and call LLM.

salience_gate

Decide whether the new turn deserves durable memory.

write_memory

Write semantic memory to Postgres and create graph links.

Important: this is where production hardening matters most. Request-path DB access must use scoped identity, graph expansion must be permission-filtered, and writes must redact sensitive data before storage.

3. Agent Architecture: Memory Storage + Retrieval

Canonical Memory Store

Semantic Memory

Stable facts, preferences, corrections, business context.

Episodic Memory

Past interactions, sessions, events, conversation history.

Procedural Memory

Skills, playbooks, reusable workflows, tool procedures.

Retrieval Engine

FTS Search

Keyword/exact-match recall.

Vector Search

Semantic similarity through embeddings.

Graph Expansion

Related entities and linked memories after permission filtering.

Retrieval Logs

Query, results, latency, trace_id, context_used.

4. SkillLoop: Post-Run Learning + Governance

AgentTrace

Completed run: messages, tool calls, runtime metadata, hashes.

Evaluation

Deterministic scoring using tool success, errors, tests, user feedback.

Distillation

Create memory and skill proposals from the evaluated trace.

Review Gate

Human approves or rejects proposed learning artifacts.

Apply / Export

Approved memory/skill files, SFT/DPO datasets, training configs.

5. SkillLoop Observational Layer

Artifact What It Observes Why It Matters
AgentTrace Messages, runtime source, adapter metadata, raw and normalized hashes. Reconstructs what happened before storing durable memory.
ToolCall Tool name, args, result, status, duration, errors, artifact refs. Separates verified tool evidence from assistant claims.
Evaluation Score, tags, notes, evidence, evaluator version. Prevents low-quality traces from becoming trusted learning.
Proposal provenance Source trace, source evaluation, producer version, code hash. Makes every memory/skill candidate auditable.
Controller run reports Ingest/eval/distill/export actions and errors per tick. Provides operational history for the governance loop.

6. Combined Target System

Live Runtime

Invoke Agent

Gateway or adapter calls LangGraph.

Retrieve Memory

Semantic + episodic + procedural context.

Use Skill

Rank procedural memory / skill and execute workflow.

Trace Everything

Events, retrievals, model calls, tool calls, errors.

Learning Governor

Evaluate Trace

Score quality and evidence.

Propose Updates

Memory candidate or reusable skill candidate.

Approval Policy

Human review for sensitive or low-confidence updates.

Promote Durable Artifact

Write typed memory, procedural skill, or dataset record.

The missing bridge: SkillLoop currently learns skills after the run. Agent Architecture retrieves memory during the run. The world-class version connects them so procedural skills are retrieved and invoked live, then improved by SkillLoop after execution.

7. What “Skill Use” Should Become

Procedural Store

Approved workflows live as typed procedural memory or SKILL.md files.

Skill Retrieval

Task intent retrieves relevant workflows before model execution.

Skill-Guided Run

LangGraph uses the selected skill as an execution plan/checklist.

SkillLoop Review

Trace outcome improves, supersedes, or rejects the skill.
